Это не ошибка в настройках, а ограничение сервиса. Оно касается не вашего стрима, а именно зрителей из России (и некоторых других стран). Зрители из других регионов всё также могут смотреть ваш стрим в установленном вами разрешении. Сама по себе ваша трансляция не ограничена по разрешению и битрейту, даже если вы ведёте её из РФ.

| Ситуация для Стримера                                                                   | Ситуация для Зрителей                                                                                                                          |
| --------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| ==**Партнёр/Компаньон**==<br>(стрим с транскодингом)<br>Есть доступ к 1440p (beta)*     | До 1440p\** для определёных стран<br>**До 1080p\** для зрителей из РФ**                                                                        |
| ==**Партнёр/Компаньон**==<br>(стрим с транскодингом)<br>Без доступа к 1440p (beta)*     | До 1080p и 8500kbps для всех, кроме РФ<br>**До 720p и 3500kbps для зрителей из РФ**<br>(даже при включении Enchanced Broadcasting)             |
| ==**Обычный Стример**==<br>(стрим с транскодингом)\*\*\*<br>Без доступа к 1440p (beta)* | До 1080p и 8500kbps для всех, кроме РФ<br>**До 720p и 3500kbps для зрителей из РФ**<br>(даже при включении Enchanced Broadcasting)             |
| ==**Обычный Стример**==<br>(стрим без транскодинга)<br>Без доступа к 1440p (beta)*      | Только ваш стрим-источник для всех зрителей          <br>**До 1080p и 8500kbps для всех**\****<br>(720p и 3500kbps при Enchanced Broadcasting) |
